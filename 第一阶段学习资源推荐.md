## 💡 学习建议

### 每周时间安排（推荐）

```
周一：看书/视频 2h + 阅读文章 1h
周二：代码实践 2h + 源码阅读 1h
周三：深入原理 2h（视频/文章）
周四：动手练习 2h
周五：综合练习/小项目 2h
周末：总结 + 复盘 + 下周规划 2h
```

### 学习方法
